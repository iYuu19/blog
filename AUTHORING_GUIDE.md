# 后台使用说明

你现在在 `/admin` 里，平时主要就看左侧这 4 个入口：

- `分类中心`
- `模板中心`
- `写文章`
- `改博客结构`
- `使用说明`

## 1. 我想发新文章，去哪

左侧点 `写文章`。

这里可以直接：

- 新建文章
- 改正文
- 改标题和摘要
- 设成草稿
- 勾选首页推荐
- 上传封面图
- 填分类、标签、比赛信息

如果你不确定分类该怎么写，建议先去 `分类中心` 看一下现有分类名和说明。
如果你不知道正文怎么起手，建议先去 `模板中心` 找一个最像的模板。

## 1.1 我想先找个模板照着写，去哪

左侧点 `模板中心`。

里面现在已经有这些：

- 通用比赛 WP 模板
- Web 比赛 WP 模板
- 电子取证 / 流量分析模板
- 赛后复盘模板
- 专题总结模板

最简单的用法就是：

1. 先看哪个模板最像你要写的内容
2. 记一下它建议的结构
3. 再回到 `写文章` 里按那个顺序写

## 2. 我想改博客名字、描述、页脚，去哪

左侧点 `改博客结构 -> 品牌和基础信息`。

这里主要控制：

- 博客名字
- 顶部的小副标题
- 网站简介
- 页脚简介
- 版权名字

## 3. 我想改顶部导航，去哪

左侧点 `改博客结构 -> 导航菜单`。

这里每一项都有：

- `名称`：页面上显示的字
- `路径`：点击后跳到哪里

常见路径示例：

- `/`
- `/contests`
- `/categories`
- `/blog`
- `/tags`
- `/about`

## 4. 我想改首页文案，去哪

左侧点 `改博客结构 -> 首页内容`。

这里可以改：

- 首页眉题
- 首页主标题
- 首页导语
- 首页介绍段落

如果你想让首页更像自己的风格，最适合先改这里。

## 5. 我想改关于页，去哪

左侧点 `改博客结构 -> 关于页内容`。

这里可以改：

- 关于页标题
- 开场介绍
- 你主要写什么内容
- 你的写作方式
- 最后一段说明

## 6. 我想新增一个自己的分类，怎么做

现在建议分两步，会更清楚：

1. 先去左侧 `分类中心`，新增一条分类说明
2. 再回到 `写文章`，把文章里的 `分类` 写成同一个名字

比如你可以先在 `分类中心` 建这些：

- 比赛 WP
- 赛后复盘
- 学习笔记
- 工具记录
- 题型总结
- Forensics
- Web 安全

这样做的好处是：

- 你自己以后不会忘记这个分类主要是干什么的
- 前台 `/categories` 会同时显示这个分类的说明
- 就算这个分类暂时还没文章，也可以先把结构搭起来

写完以后：

- `/categories` 会自动出现这个分类
- 这个分类页会自动生成
- 首页里的分类区域也会自动跟着更新

## 7. 比赛名和比赛标识怎么填

如果几篇文章属于同一场比赛：

- `比赛名`：写给人看的名字
- `比赛标识`：写给链接用的英文或拼音短名

例子：

- 比赛名：`黄河流域公安院校网络安全技能挑战赛`
- 比赛标识：`yellow-river-police-2026`

这样前台就能自动聚合成一个比赛页。

## 8. 草稿和首页推荐怎么用

- `草稿 = true`：后台能看到，前台不会公开显示
- `草稿 = false`：前台正常公开
- `首页推荐 = true`：这篇更容易被放到首页重点区域

## 9. 我写文章时建议先填什么

最常用的一套顺序：

1. 先写标题
2. 再写摘要
3. 选发布时间
4. 填分类
5. 填标签
6. 如果是比赛题，再补比赛名和比赛标识
7. 最后写正文

## 10. 正文怎么写更清楚

可以直接按这个顺序写：

1. 背景
2. 题目入口
3. 分析过程
4. 关键命令 / payload / 截图
5. 结论
6. 复盘

## 10.1 我想像 Word 一样边写边插截图

现在 `写文章 -> 正文` 已经改成更接近 Word 的模式了，推荐这样用：

1. 先把光标放到你想插图的位置
2. 直接 `Ctrl + V` 粘贴截图
3. 或者把图片拖进正文区域
4. 需要代码时，工具栏里的 `</>` 是行内代码，`CB` 点下去会先让你输入语言，再自动插入带语言的代码块
5. 图片插进去以后，继续在图片前后写分析过程

如果你懒得自己补语言名，可以直接点下面那排快捷按钮：

- `JS`
- `Python`
- `Bash`
- `HTML`
- `YAML`
- `JSON`
- `SQL`
- `Text`
- `自定义语言`

这些按钮会直接插入带语言名的代码块，前台会自动按语言做语法高亮。文章页里的代码块右上角还会有 `复制代码` 按钮。

最顺手的写法就是：

- 先写一小段说明
- 紧跟一张截图
- 再补这张图说明了什么

这样写出来的效果会比单独堆一大段文字更像题解，也更接近平时在 Word 里排版。

## 11. 想做得更有自己的风格，可以先改哪几处

最容易出效果的地方有这几个：

- `品牌和基础信息` 里的博客标题和副标题
- `首页内容` 里的首页主标题和导语
- `导航菜单` 里的栏目名字
- 文章里的 `分类`

比如你可以把分类慢慢整理成自己的结构：

- 比赛 WP
- 赛后复盘
- 学习笔记
- 工具记录
- 取证练习
- Web 题型
- Misc 题型

## 12. 忘了怎么改时怎么办

先看你想做的是哪一种：

- 想发内容：点 `写文章`
- 想先找一份写法：点 `模板中心`
- 想改外观和结构：点 `改博客结构`
- 想确认规则：点 `使用说明`
