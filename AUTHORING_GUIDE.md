# 后台使用说明

你现在主要看左侧这些入口：

- `仪表盘`
- `文章中心`
- `页面文案`
- `板块`
- `导航`
- `分类与标签`
- `全站设置`
- `媒体库`
- `写作模板`
- `使用帮助`

## 1. 想发新文章去哪里

去 `文章中心`。

这里可以直接：

- 新建文章
- 改草稿
- 改标题和摘要
- 设为草稿或公开
- 勾选首页推荐
- 上传封面图
- 填分类、标签、比赛信息

## 2. 想先找一个模板照着写

去 `写作模板`。

现在已经有：

- 通用比赛 WP 模板
- Web 比赛 WP 模板
- 电子取证 / 流量分析模板
- 赛后复盘模板
- 专题总结模板

最顺手的方式就是：

1. 先找一个最像的模板
2. 看它的结构
3. 再回到 `文章中心` 按那个顺序写

## 3. 想改前台固定页面文案

去 `页面文案`。

这里可以改：

- 首页文案
- 关于页文案
- 留言板文案
- 归档页文案
- 分类页文案
- 标签页文案
- 详情页固定文案
- 404 页面文案

## 4. 想改首页结构或区块顺序

去 `板块`。

这里主要负责：

- 首页板块顺序
- 首页板块显示隐藏
- 首页主栏 / 侧栏位置
- 首页自定义富文本块
- 关于页、归档页、分类页、比赛页、标签页这些页面的区块顺序

### 想新增首页说明块

去 `板块 -> 首页板块与公共区`：

1. 在 `首页板块编排` 里新增一项
2. 类型选 `自定义富文本块`
3. 填眉题、标题、说明和正文
4. 选择主内容列或侧栏列

## 5. 想改顶部导航

去 `导航 -> 导航菜单`。

每一项都只有两个核心字段：

- `名称`
- `路径`

常见路径示例：

- `/`
- `/contests`
- `/categories`
- `/blog`
- `/tags`
- `/about`

## 6. 想补分类或标签说明

去 `分类与标签`。

这里可以维护：

- 分类说明卡
- 标签说明卡
- 示例主题
- 排序
- 关联分类

好处是：

- 前台 `/categories` 和 `/tags` 会更完整
- 详情页会有说明，不只是单纯文章列表
- 就算暂时没文章，也能先把结构搭起来

## 7. 比赛名和比赛标识怎么填

如果几篇文章属于同一场比赛：

- `比赛名`：写给人看的名字
- `比赛标识`：写给链接用的英文或拼音短名

示例：

- 比赛名：`黄河流域公安院校网络安全技能挑战赛`
- 比赛标识：`yellow-river-police-2026`

这样前台就能自动聚合成比赛页。

## 8. 草稿和首页推荐怎么用

- `草稿 = true`：后台能看到，前台不公开
- `草稿 = false`：前台正常公开
- `首页推荐 = true`：更容易被放到首页重点区域

## 9. 正文建议怎么写更清楚

最常用顺序：

1. 背景
2. 题目入口
3. 分析过程
4. 关键命令 / payload / 截图
5. 结论
6. 复盘

## 10. 图片应该放哪里

去 `媒体库` 看目录说明。

现在主要是：

- `public/uploads`
  - 正文截图、步骤图、普通配图
- `public/contest-covers`
  - 比赛封面、比赛索引卡、比赛详情视觉图

### 最顺手的上传方式

在 `文章中心` 里边写边粘贴截图或拖拽图片，正文图片一般会直接进入 `public/uploads`。

## 11. 想边写边看阅读效果

正文编辑器里现在有一个 `Show Preview / Hide Preview` 按钮。

它会在编辑器下方显示一张实时阅读预览卡，能一起看到：

- 标题
- 摘要
- 分类 / 标签
- 比赛信息
- 封面图
- 正文渲染效果

这样写长文时，不用频繁切到前台确认排版。

## 12. 想改全站基础信息

去 `全站设置`。

这里可以改：

- 网站标题
- 头部副标题
- 网站描述
- 页脚描述
- 头像
- 默认分享图
- 默认封面
- 社交链接
- 评论系统参数

## 13. 忘了去哪改怎么办

先判断你现在要改的是哪一类：

- 发文章：`文章中心`
- 改固定页面文字：`页面文案`
- 改首页和索引页结构：`板块`
- 改顶部菜单：`导航`
- 补分类标签资料卡：`分类与标签`
- 看图片目录规则：`媒体库`
- 改站点基础设置：`全站设置`
