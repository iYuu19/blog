# 模板怎么用

后台里现在有一个 `模板中心`。

建议每次发文章都按这个顺序来：

1. 先点左侧 `模板中心`
2. 找一个最像你这篇内容的模板
3. 看它建议的结构和字段
4. 再回到左侧 `写文章`
5. 按那个模板去填标题、分类、方向、正文

## 什么时候选哪种模板

- `通用比赛 WP 模板`
  适合大多数单题题解
- `Web 比赛 WP 模板`
  适合 Web 题，从入口到漏洞利用链都能按它写
- `电子取证 / 流量分析模板`
  适合样本分析、证据链、时间线这类题
- `赛后复盘模板`
  适合写一整场比赛，不只是单题
- `专题总结模板`
  适合长期整理某个知识点、工具或题型

## 最简单的理解方法

- 题解：优先看 `比赛 WP` 类模板
- 整场总结：看 `赛后复盘模板`
- 长期笔记：看 `专题总结模板`

## 模板不是死规则

模板只是让你起步更轻松，不是必须一模一样照抄。

如果某一节不适合这篇文章，可以直接删掉；
如果你有自己想加的内容，也可以继续往下补。
