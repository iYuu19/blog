# 站点内容中枢

这个后台现在不只是发文章，也是在逐步接管整个前台站点的主要内容和结构。

## 左侧菜单现在怎么分

- `仪表盘`
  - 先看这里，快速了解后台分工和推荐操作顺序。
- `文章中心`
  - 日常发文章、改草稿、设首页推荐、补比赛信息，都在这里。
- `页面文案`
  - 首页、关于页、留言板、归档页、分类页、标签页、404 页这些固定页面文案都在这里维护。
- `板块`
  - 首页板块顺序、首页侧栏、自定义富文本块、各索引页区块顺序都在这里调整。
- `导航`
  - 顶部导航菜单在这里维护。
- `分类与标签`
  - 分类说明卡、标签说明卡、示例主题、排序都在这里维护。
- `全站设置`
  - 网站标题、头像、社交链接、默认分享图、评论系统参数都在这里改。
- `媒体库`
  - 图片目录、命名方式和上传建议都集中放在这里说明。
- `写作模板`
  - 发文前先挑模板，减少漏项。
- `使用帮助`
  - 忘了去哪改、字段怎么填时，先看这里。

## 推荐操作顺序

1. 先在 `文章中心` 写文章。
2. 如果前台某一页文案不顺，就去 `页面文案` 改文字。
3. 如果想调整首页和索引页结构，就去 `板块`。
4. 如果想改顶部菜单，就去 `导航`。
5. 如果想让分类页和标签页更完整，就去 `分类与标签` 补说明卡。
6. 如果要统一整理截图和封面，就去 `媒体库` 看目录规则。

## 常见场景

### 想在首页新增一块说明内容

去 `板块 -> 首页板块与公共区`：

- 在 `首页板块编排` 里新增一项
- 类型选 `自定义富文本块`
- 填 `眉题 / 标题 / 说明文字 / 富文本正文`
- 选择放在主内容列还是侧栏列

### 想补一个还没发过文章的分类或标签

直接去 `分类与标签` 里先建说明卡。前台会先有入口，后面再慢慢补文章。

### 想知道图片应该传到哪里

去 `媒体库` 看目录说明：

- 正文截图一般进 `public/uploads`
- 比赛封面一般进 `public/contest-covers`

## 当前内容还是怎么存

- 文章：`src/content/blog/*.md`
- 页面文案：`src/content/site-*.yaml`
- 板块结构：`src/content/site-layout/*.yaml` 和 `src/content/site-page-layouts/*.yaml`
- 分类标签说明：`src/content/site-categories/*.yaml` 和 `src/content/site-tags/*.yaml`
- 媒体说明：`src/content/site-media/*.yaml`
- 图片：`public/uploads/*` 和 `public/contest-covers/*`

## 这版后台的原则

- 保留文件化内容，方便 Git 管理和回滚
- 先做受控区块，不做完全自由搭建器
- 先把常用页面和常用结构接进后台，再逐步扩展
