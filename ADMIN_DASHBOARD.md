# 站点内容中枢

这个后台现在不只是发文章，也可以逐步管理前台的主要结构和文案。

## 快速入口

- `文章中心`：新建、编辑、草稿、首页推荐、封面图、标签都在这里。
- `页面文案`：首页、关于页、留言板、归档页、分类页、标签页、404 页的主要文案都在这里改。
- `结构与导航`：导航菜单、首页板块顺序、首页侧栏内容、各索引页区块顺序都在这里调。
- `分类与标签`：分类说明卡、标签说明卡、示例主题和排序都在这里维护。
- `全站设置`：站点标题、头像、社交链接、默认分享图、评论系统参数都在这里改。

## 推荐编辑顺序

1. 先在 `文章中心` 写文章。
2. 如果前台某一页文案不顺，就去 `页面文案` 改文字。
3. 如果想调整首页和索引页长什么样，就去 `结构与导航` 改板块顺序和显示。
4. 如果想让分类页、标签页看起来更完整，就去 `分类与标签` 补说明卡。

## 常见场景

### 想新增首页上的一块说明内容

去 `结构与导航 -> 首页结构与全站公共区`：

- 在 `首页板块编排` 里新增一项
- 类型选 `自定义富文本块`
- 填 `眉题 / 标题 / 说明文字 / 富文本正文`
- 选择放在主内容列还是侧栏列

### 想修改分类页或标签页前台介绍

去 `页面文案` 修改对应页面文案，再去 `分类与标签` 补充具体分类或标签说明。

### 想补一个还没写过文章的分类或标签

可以先在 `分类与标签` 里提前建好说明卡。前台会先显示这个入口，后面再慢慢补文章。

## 现在的内容存放方式

- 文章：`src/content/blog/*.md`
- 页面文案：`src/content/site-*.yaml`
- 板块结构：`src/content/site-layout/*.yaml` 和 `src/content/site-page-layouts/*.yaml`
- 分类标签说明：`src/content/site-categories/*.yaml` 和 `src/content/site-tags/*.yaml`
- 图片：`public/uploads/*`

## 这一版的原则

- 保留文件化内容，方便 Git 回滚
- 先做受控区块，不做完全自由搭建
- 先把常用页面和常用结构都接进后台，再逐步扩展
